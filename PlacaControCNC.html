Placa Interface Controladora via Internet (Ethernet Cabo RJ45) ou Wifi para Software Mach3, Router CNC, Fresadora, Torno, Impressora 3D. Essa placa é responsável por fazer a conexão entre seu PC (CPU) e sua máquina computadorizada (CNC). A vantagem dessa Interface é a conexão via Internet e Wifi entre PC e sua máquina CNC. Não utiliza cabo paralelo / DB25 / serial. A conexão é feita com cabo de rede tipo RJ45 para conexões em rede interna LAN ou externa via Internet. Ou ainda Wifi. Placa completa, suporta até 6 eixos com 200kHz de transmissão de dados em cada eixo. Placa de alta performance para extrair o máximo de rendimento. Possui entradas de sinal podendo ser habilitado: Botão de Emergência ESTOP, Sensores Home, Sensores Fim-De-Curso, Sensor Apalpador, Sensor TouchProbe, Sensor de Zeramento de Ferramenta (AutoToolZero). Possui saídas digitais para ligar / desligar Relé (Relay) podendo ser habilitado: Ligar / Desligar Spindle (comandos M3 e M4) sentido de giro horário / anti-horário (conforme opções disponíveis no inversor) Ligar / Desligar Refrigeração: Flood / Mist Marca: Novosun / NVEM Modelo: NOVOSUN NVEM 6AX ETHERNET WIFI Tipo: Controladora / Isoladora Software de compatibilidade: Mach3 Conexão de dados: via cabo de rede / RJ45 ou Wifi. Conexão de energia: 12V DC Conexão paralela / DB25: Não Quantidade de eixos: 6 >> X Y Z A B C Saídas Analógicas: 1 saída 0-10V PWM / controle de RPM DIMENSÕES: 140 X 80 [mm] ACOMPANHA: 1 PLACA INTERFACE CONTROLADORA MACH3 1 CABO DE REDE TIPO LAN RJ45/RJ45 BLINDADO
